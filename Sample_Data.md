## 3. fix-script.sh

```bash
#!/bin/bash

sudo ufw enable
sudo systemctl enable fail2ban
sudo systemctl enable auditd
sudo sed -i 's/^PermitRootLogin yes/PermitRootLogin no/' /etc/ssh/sshd_config
sudo systemctl restart ssh

echo "Recommended hardening actions completed."
```

### Purpose

Generated remediation script containing recommended hardening actions.
